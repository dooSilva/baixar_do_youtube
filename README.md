# Baixador de musicas do youtube

## 🚀 Tecnologias
- C#
- Sqlite

## 📦 Como rodar o projeto
-Dentro de baixar_do_youtube crie uma pasta chamada bin e dentro dela outra pasta Debug: bin/Debug

-Faça o download do ffmpeg: https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl-shared.zip

-Descompacte tudo 

-Dentro de ffmpeg/bin pegue todos os arquivos dll e copie tudo para a pasta bin/Debug do baixar_do_youtube

-Faça o download do https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe

-Copie o programa yt-dlp.exe e também cole na pasta bin/Debug do baixar_do_youtube

-No visual studio em Program: coloque seu endereço MAC da sua placa de rede ethernet

-Compile e pronto!
